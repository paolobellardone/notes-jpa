Simple CRUD Notes

This application demonstrates how a Helidon application implements a CRUD endpoint to manage simple notes. This simple frontend interacts via REST APIs with the Helidon Backend.

Behind the scenes, we have:

  • EclipseLink ORM taking care of all CRUD operations
  • JAX-RS powering the REST API
  • CDI based dependency injection framework
  • HikariCP for datasource implementation
  • Kubernetes as a Container Platform
  • Jenkins for CI/CD
  • Oracle for Database persistence
